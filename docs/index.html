Introduction to microbiome data science

Introduction to microbiome data science

Leo Lahti and Henrik Eckermann

2019-09-13

1 Overview

This course provides an overview of the standard bioinformatics workflow in taxonomic profiling studies, ranging from data preprocessing to statistical analysis and reproducible reporting, with a focus on examples from human gut microbiota studies.

Target audience Advanced students and applied researchers who wish to develop their skills in microbial community analysis.

Venue Radboud University, Nijmegen. Sep 12-13, 2019, organised in collaboration with University of Turku, Finland.

Learning goals The students will become familiar with standard bioinformatics concepts and methods in taxonomic profiling studies of the human microbiome. This includes better understanding of the specific statistical challenges, practical hands-on experience with the commonly used methods, and reproducible research with R. After the course the students know how to approach new programming tasks by utilizing available documentation and R tools.

Reusing the material Unless otherwise stated, the teaching materials can be freely used, modified and shared based on the open 2-Clause BSD License. Kindly cite the material as follows: “Lahti Leo, Henrik Eckerman, Sudarshan Shetty (2019). Introduction to microbiome data science. URL: https://microbiome.github.io”

1.1 Preparing for the course

1.1.1 Software setup

We recommend using your own laptop. If this is not possible, kindly contact the organizers.

Follow the below steps before the course:

  1. Install R: Follow the Download link and then choose your operating system. Then in the next step choose R-3.6.1 for Mac or “base” and then R 3.6.1 for Windows. If you already have installed a recent version of R, install Rstudio (see step 2) and then go to step 3. In case installing or loading the packages in step 3 fails, consider updating R by performing step 1. We tested the material using the newest version (R 3.6.1) as well as R 3.5.
  2. Install/Update Rstudio: Follow the Download link and choose “Rstudio Desktop” (Open Source License) to download the newest version of Rstudio.
  3. Open this Installation script in Rstudio and run the entire script. This will install all required packages for the course. It might be that you have to respond to messages in the R console. E.g. R might ask you if you want to update all, some or no packages for the installation. Then you can type “a” for “all” and enter. The installation of the script might take 5-20 minutes depending on your machine. The installation finished when the “>” appears again in the R console. You can test if the installation worked by trying to load the packages like this:
# check if all packages can be loaded
library("tidyverse")
library("vegan")
library("phyloseq")
library("rmarkdown")
library("knitr")
library("BiocManager")
library("devtools")
library("DESeq2")
library("devtools")
library("microbiome")
library("microbiomeutilities")

If you are not familiar with R programming, we highly recommend going through Data Analysis and Visualization in R for Ecologists by Data Carpentry and further links to R resources in the study material.

For questions regarding the installation of the software you may contact Henrik.
