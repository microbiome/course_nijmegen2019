Introduction to microbiome data science

Introduction to microbiome data science

Leo Lahti and Henrik Eckermann

2019-08-10

1 Overview

The course provides an overview of the standard bioinformatics workflow in taxonomic profiling studies, with a focus on human gut microbiota, ranging from data preprocessing to statistical analysis and reproducible reporting.

Learning goals Introduce microbial community data analysis. There will be talks and discussion on theory and methodology for analysis of microbial community data. We will cover topics ranging from design of studies, sequencing technologies, importance of controls and standardized DNA processing. Supervised hands-on training covering analyses from raw reads using NG-Tax, downstream analysis in R for exploration and analysis of microbiome sequencing data will be priority. There will be a strong focus on using R, R Studio for moving towards reproducible and open science.

Target audience Anyone who plans to or is currently doing high throughput microbial community analysis.

Venue This course will be held at Radboud University, Nijmegen on Sep 12-13, 2019, organised in collaboration with University of Turku, Finland.

1.2 Reusing the material

Unless otherwise stated, the teaching materials can be freely used, modified and shared based on the open 2-Clause BSD License. Kindlty cite the material as follows: “Lahti Leo, Henrik Eckerman, Sudarshan Shetty (2019). Introduction to microbiome data science. URL: https://microbiome.github.io”

1.2.1 Background reading

The following background materials can help to get familiar with the course topics in advance.

  • R programming basics: Base R
  • R visualization: ggplot2 and R graphics cookbook
  • Reproducible document generation with R Markdown
  • RStudio IDE
  • Microbiome bioinformatics in R: List of R tools for microbiome analysis. See in particular phyloseq and microbiome tutorial.

  • Data Analysis and Visualization in R for Ecologists by Data Carpentry

  • [Metagenomics meets time series analysis: unraveling microbial community dynamics. Faust K. Current Opinion in Microbiology, 25:56–66, 2015]

  • [Microbial communities as dynamical systems. Gonze D et al. Current Opinion in Microbiology, 44:41–49, 2018]

  • Microbiome Data Science. Shetty & Lahti, 2019
