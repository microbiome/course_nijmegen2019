4 Hands-on 2

4.1 Key concepts: core, diversity, community variation

Next, let us familiarize with ecological concepts and methods for analysing microbiome profiling data.

  • Core microbiota Pick up the core microbiota including taxa that exceed 0.1% relative abundance in over 50% of the samples (prevalence). How many core taxa there are? You can read more about the core microbiota definition in Salonen et al. 2012

  • What is the most prevalent genus in the data (tip: microbiome::prevalence and sort)

  • Special properties of microbiome data: sparsity, compositionality, non-Gaussianity. Visualize the population distribution of abundances for the following taxa: Akkermansia, Dialister, Gemella. Do the same for CLR-transformed abundances.

  • Visualize the core microbiota (examples)

  • Alpha diversity Estimate taxonomic diversity for each sample (function microbiome::alpha). Compare the results between two or more diversity indices.

  • Beta diversity Visualize community variation with PCoA (examples).

  • Use PERMANOVA to investigate whether the community composition differs between obese and non-obese individuals. You can also include covariates such as age and gender, and see how this affects the results.

4.2 Special properties of microbiome data

  • Influence of the data transformations on statistical analysis Visualize community variation with PCoA with the following options: 1) Bray-Curtis distances for compositional data; 2) Euclidean distances for CLR-transformed data.

  • Differential abundance analysis: use DESeq2 to identify which genera are associated with gender differences (examples on DESeq2). For more explanation on the method, see DESeq2 R/Bioconductor. Note that controlling key covariates (diet, medication, age..) may have remarkable influence on data interpretation (Falony et al. Science 2016)
